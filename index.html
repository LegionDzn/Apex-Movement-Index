Apex Legends runs on a heavily modified version of the Titanfall 2's Source engine, which itself is a modification of the Titanfall 1's Source engine. This layered and adapted version of the source engine creates some unintended side effects and glitches, many of which we abuse to gain the upper hand. Many movements techniques abuse these unintended features to allow us to do things the developers never intended. When we use these unintended features to change the way we move about the game world, we call them movement tech, although some of the movement techniques within apex legends are fully intended, many are not, which results in them being very unintuitive. For this reason, I've created Apex Movement Index, a website that attempts to document and catalog the many various movements techniques Within Apex Legends.

This website is ever-changing and still in development. So there are some incomplete pages. I plan to document all known interactions, techniques, and tricks which pertain to movement in Apex Legends, and perhaps other tips and tricks which may be unintuitive to the average player.

I know the pages are wordy and not that appealing to the eyes, but I've found that this is the least convoluted way to catalog and explain the tech without using multiple-minute-long videos.
